## Description
  FanScrapr is a simple web scrapper that scrapes instagram pages for images



### 
  How to use
    Download the zip file or pull
  - open a terminal in the directory 
  - run: npm install 
  - run: node app 

  A prompt will appear to take details 



#### Technologies
- node js
- puppeteer
